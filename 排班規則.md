# 排班規則

## 每日最低人力需求

| 職位 | 人數 | 備註 |
|------|------|------|
| 廚房 | 1 人 | |
| 吧台 | 1～2 人 | |
| 外場 | 2 人以上 | PT（21-2）計入外場 |
| **合計** | **4 人以上** | |

> 五人以上出勤時，若外場僅 1 人，可由備班補足第 2 個外場名額。  
> 五人以上且外場已 ≥ 2 人、廚房無人時，可由備班補廚房缺口。

---

## 班別分配優先順序

| 職位 | 優先順序 |
|------|---------|
| 廚房 | Teen → Susan → 小黃 |
| 吧台 | Eric → 阿維 → 小黃 |
| 外場 | Susan → 小黃 → Sharon → Sue → 子芯 |
| 備班 | Susan → 阿維（有備班技能者）→ Sharon |

> 吧台第 2 人須在外場已達 2 人以上後才排入。

---

## 休假規則

- 每月原則 **8 天**休假
- **週二**為固定公休，自動列入休假（不計入 8 天）
- **國定假日**視為普通工作日排班，若休假配額有剩且人力符合需求，可排休
- 特別休假（特休）另行計算，不併入每月 8 天

### 例外人員

| 人員 | 規則 |
|------|------|
| Eric、Teen、Sue | 不受每月 8 天休假限制 |
| 子芯（PT） | 休假一律手動處理 |

## 注意事項

- 週二為固定公休日
- 禮拜五、六為最忙碌時期，請盡可能全員到齊
- 備班人員可視當日需求支援外場或廚房，出勤當天須 **5 人以上**
